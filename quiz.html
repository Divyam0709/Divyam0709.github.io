Quiz

Questions

  1. What do you mean by Convolutional Neural Network?
  2. What do you mean by Convolution?
  3. What is difference between VGG 16 and VGG 19?
  4. Image Classification is example of which type of Deep Learning?
    • Supervised
    • Unsupervised
    • Semi Supervised
    • Reinforcement learning
  5. Which deep learning model is best for Image Classification?
    • DNN (Deep Neural Network)
    • CNN (Convolutional Neural Network)
    • RNN (Recurrent Neural Network)
    • MNN (Modular Neural Network)

Answers

  1. A convolutional neural network (CNN) is a type of artificial neural network used primarily for image recognition and processing, due to its ability to recognize patterns in images.
  2. Convolution is a general purpose filter effect for images. It is a matrix applied to an image and a mathematical operation comprised of integers.
  3. The concept of the VGG19 model (also VGGNet-19) is the same as the VGG16 except that it supports 19 layers. The “16” and “19” stand for the number of weight layers in the model (convolutional layers). This means that VGG19 has three more convolutional layers than VGG16.
  4. Supervised
  5. CNN (Convolutional Neural Network)
